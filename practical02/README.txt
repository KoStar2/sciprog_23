This file contains:

Conversion.c and Conversion.f90:
C and FORTRAN programs that
take an integer input and convert it to binary.

Sum.c and Sum.f90:
C and FORTRAN programs that calculate the series 1/n
from 1 to 1000 starting at 0 and starting at 1000,
then calculates the difference between the values
calculated by the two methods.
