This file contains:

finde.c:
A program that estimates the value of e
using a taylor series approximation with x = 1

pointers.c:
A program that contains various functions for pointer-operations on arrays;
allocatearray(int n), which returns a pointer to an integer array of size n;
fillwithones(int *n), which take a pointer to an integer array and fills the array with ones;
printarray(int *n), which takes a pointer to an integer array and prints the array;
deallocatearray(int *n) which takes a pointer to an integer array and frees the memory held.
