This file contains:

main.c:
A program that asks the user for a text file containing
a square matrix, then prints if the matrix is a magic square
or not. A magic square is a square matrix where every row, column,
and diagonal sum to the same value.

magic_square.txt:
an example magic square for testing.

not_magic_square.txt:
an example matrix that is not a magic square for testing.

magic_square.h:
A header file containing a function that determines if
a square matrix is magic, and returns 1 (TRUE)  if it is, and 0 (FALSE) if it is not.
